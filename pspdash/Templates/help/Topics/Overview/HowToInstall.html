How To Install

System Requirements

The Process Dashboard is 100% Java, and will run on any system with a Java 2 compliant virtual machine. This includes Windows, Linux, and Unix systems. (We haven't yet had an opportunity to test it on a Macintosh system, but don't anticipate any problems; we would love to hear from a Macintosh user who would like to give the dashboard a try.)  For access to forms and scripts, a web browser is also required.

To use the process dashboard, you must first install the following software:

  • The Java Runtime Environment, version 1.2 or greater (available from Sun Microsystems)
  • One of the following web browsers (Internet Explorer is preferred).
    • Internet Explorer 4.0 or greater (available from Microsoft).  If you choose to perform a custom installation of IE, make certain to install the "Data Binding" option.
    • Netscape 4.7 (available from Netscape).    Note: Netscape 6.0 is not currently compatible with the dashboard.
  • Make certain that Java and JavaScript are enabled in your browser. (This is typically the default setting, so you usually don't need to worry about enabling these unless you explicitly disabled them in the past.)
  • If you connect to the internet via a proxy server, you may need to tell your browser to bypass the proxy for the address "localhost".  The instructions below may not apply to your browser word-for-word, but should hopefully get you pointed in the right direction:
    • For Internet Explorer, choose "Tools > Internet Options". Click on the "Connections" tab, then on the "LAN Settings" button.  Make certain the "Bypass proxy server for local addresses" box is checked.
    • For Netscape, choose "Edit > Preferences".  Under the Category "Advanced > Proxies",  click the "Manual proxy configuration" button, and ensure that the "No Proxy for" box contains "localhost".

Installation Instructions:

Before attempting to install the dashboard, make certain you have installed the required software listed above.  Then choose the installation process that describes your system:

 
  • If you have previously installed the dashboard, follow the upgrade instructions.
  • If you are running Windows, and your default web browser is one of the web browsers listed above, choose the simple windows installation.
  • If you are running Windows, and your default web browser is Netscape 6.0, choose the custom windows installation.
  • If you are running Linux or some other flavor of Unix , choose the unix installation.
  • If you are running MacOS, contact us.  We would love to help you install the software on your system, so we can learn what is involved and publish our findings on this website.

Add-on Process Sets

The following processes do not come with the dashboard by default, but are easy to install:

  • SCR/DCR - a set of processes (inspired by PSP3) created by the Hill AFB TaskView team for software development.  (These processes were distributed with version 1.1 of the dashboard, but are no longer actively supported.)

To install an add-on process set:

  • Locate the file " pspdash.jar" on your system.  (If you are on a Windows system, this file is most likely located in the directory "C:\Program Files\Process Dashboard".)
  • Click on a link above to download the add-on process set. When your browser asks you whether you want to save the file or open it; choose Save.  Save the file into the directory you found in the previous step. (In other words, the pspdash.jar file and the add-on process file should be side-by-side in the same directory.  Do not overwrite the pspdash.jar file, just save the process file into the same directory.)
  • If the dashboard is running, you will need to restart it for the new processes to be found.

Upgrade to New Version

  1. If the dashboard is currently running, stop it first.
  2. Locate the file "pspdash.jar" on your system. (If you are on a Windows system, this file is most likely located in the directory "C:\Program Files\Process Dashboard".)
  3. Download the new version of the dashboard by clicking on this link and following the directions. When your browser asks you whether you want to save the file or open it; choose Save. Save the file to the location you found in step 2, overwriting the existing "pspdash.jar" file.
    Note - this is one time you do want to overwrite the "pspdash.jar" file. Resist the temptation to rename the existing "pspdash.jar" file before saving the new one. (Click here if you want to know why.)
  4. You can then run the dashboard as you normally would.
  5. Please take a moment to read over the release notes for the new version. Please consider subscribing to our announcements mailing list. We will send email to this mailing list when new versions of the tool are available.

  3. Please consider subscribing to our announcements mailing list. We will send email to this mailing list when new versions of the tool are available.

Custom Windows Installation

Netscape 6.0 is currently not capable of supporting the process dashboard. If you are a Netscape 6.0 user, don't worry - you don't need to give up your favorite browser. You still have two options for running the dashboard:

  • Netscape will allow you to have several versions of Communicator installed on your computer without conflict. So you can install Netscape 4.7 on your computer for use with the dashboard, and keep using Netscape 6.0 as your system-default web browser.
  • Or, you can avoid installing multiple versions of Netscape by using Internet Explorer with the dashboard while keeping Netscape 6.0 as your system default. (Unless you really dislike IE, this may be the simplest option. And for what it's worth, IE does work better with the dashboard than Netscape.)

Choose between the options above, and make certain you have an appropriate browser installed on your system. Then follow the instructions below:

  1. Download and run the Process Dashboard Installer by clicking on this link and following the directions. The installation wizard will prompt you first for the Destination Location (the directory where the dashboard program will be installed), then it will prompt you for the Data Folder (which defaults to C:\pspdata).
  2. After installation is complete, create a file in the Data Folder called pspdash.ini, and add one of the following lines to that file:

    browser.command=C:/Program Files/Internet Explorer/IExplore.exe

    if you want the dashboard to display scripts and forms in Internet Explorer (recommended), or

    browser.command=C:/Program Files/Netscape/Communicator/Program/Netscape.exe

    to use Netscape. NOTE: make certain to edit the browser path (the part in italics above) so it correctly points to the executable for the browser you wish to use. To separate directory names, use either a forward slash ("/") or a double backslash ("\\").
  3. You can then run the dashboard by clicking on the "Process Dashboard" desktop icon.
  4. Please consider subscribing to our announcements mailing list. We will send email to this mailing list when new versions of the tool are available.
  5. You might want to consider taking the quick start tour.

Linux / Unix Installation

To install the process dashboard on a unix system, follow these steps:

  1. Make certain you have successfully downloaded and installed a Java Runtime Environment. Your runtime environment needs to conform to the Java Standard Edition 1.2.2 or higher, and needs support for "Swing" components.
  2. Download and install Netscape 4.7. (It may be possible to use the dashboard with Internet Explorer on unix systems, but we haven't ever done it before...) Make certain that 'netscape' is in your path.
  3. Download the file pspdash.jar (following the directions on this page) and save it on your system.
  4. Create a directory where you would like your personal process data (e.g., metrics, defects, time log, etc) to be stored.
  5. Create a shell script to run the dashboard and place it in your path. The script should look something like this:

    #!/bin/sh

    cd ~/path/to/your/personal/data/directory
    /path/to/jre/bin/java -jar /path/to/pspdash.jar
  6. Please consider subscribing to our announcements mailing list. We will send email to this mailing list when new versions of the tool are available.
  7. You might want to consider taking the quick start tour.

Upgrade Warning

When it starts up, the dashboard automatically searches the directory containing the "pspdash.jar" file for other files ending with the ".jar" extension. Any files it finds that match this pattern are interpreted as add-on process sets, and imported into the tool.

Therefore, if you decide you want to save an old "pspdash.jar" file by renaming it, make certain that it no longer ends with ".jar". Otherwise, older versions of the built-in processes may interfere with the new versions that come with the dashboard.

The easiest way to avoid all this trouble is just to overwrite the old "pspdash.jar" file with the new version you download.
