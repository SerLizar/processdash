Create a Team Project (Team Leader)

Creating a team project is simple, thanks to a Team Project Setup Wizard included in the Tools for Team Leaders add-on. To create a new team project, follow these steps:

  1. Start the Team Dashboard using the appropriately named shortcut on the team leader's computer. The Team Dashboard will open. (If your personal dashboard is also running, you can easily distinguish the Team Dashboard because it displays "Team Dashboard" in the title bar, and displays a different icon for the window.)
  2. On the C menu of the team dashboard, choose the Hierarchy menu item. The Hierarchy Editor window will appear.
  3. In the Hierarchy Editor, highlight a node in the hierarchy such as the top-level Project node. On the Edit menu, choose Add Template → Team Project. A new node called Team Project will be created; give it a short name that describes your project. Choose File → Save, then File → Close.
  4. Using the dashboard Hierarchy Menus, navigate to the project you just created. Click the dashboard Script Button to display the Team Project Setup Wizard.
  5. The Team Project Setup Wizard will walk you through the steps involved in creating a team project. Read each page carefully and supply the information requested. One page of the wizard will ask whether you wish to create a team project or a master project; choose the Team Project option.
  6. When the wizard finishes, you can close the browser window.
