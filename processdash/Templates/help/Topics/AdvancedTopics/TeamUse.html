Team Use

The process dashboard includes functionality which can be used by the individuals on a software project team.

Different teams have different needs. When using the dashboard to plan and track your project, there are at least three different approaches you could take. These approaches (from simplest to more advanced) are:

  1. Rollup Earned Value Only
  2. Rollup PSP Data and Earned Value
  3. Rollup Custom Process Data and Earned Value
