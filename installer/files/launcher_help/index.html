The Process Dashboard Enterprise Server provides a secure repository for team and personal metrics data, and it allows you to view that data by clicking hyperlinks in your web browser. When you click a dataset link on this site, the data will open in the Process Dashboard or Team Dashboard application, as appropriate.

If you have used the Process Dashboard in the past, it's likely that your computer already contains the software you need to open dashboard datasets.

If you haven't used the dashboard before, or if you click a link on this website and your computer is unable to open the file, please follow the instructions below.

Step 1: Install Java on your computer, if it is not present already. The dashboard requires Java 6 or higher; newer versions are recommended. If Java is not already installed on your computer, you can download an installer by visiting https://www.java.com/.

Step 2: Download the Process Dashboard Launcher using one of the links below, and double-click the file to install:

  • Windows
  • Mac OS X / Unix / Linux

After installing this software, you can click links on this website to open the corresponding datasets.
