PROBE Estimating Script

Phase
Number		 Purpose To guide the PROBE estimation process.
  Entry Criteria
  • Requirements statement
  • Size Estimating Template and instructions
  • LOC per method data for object types
  • Time recording Log
  • Historical size and time data
1 Conceptual Design
  • Review the requirements and produce a conceptual design to meet them.
2 Estimate Objects Follow the Size Estimating Template Instructions to estimate the new object LOC and new reused LOC.
3 Other Program LOC Follow the Size Estimating Template instructions to estimate the base, deleted, modified, added, reused, and new reused LOC.
4 The Size Estimating Procedure
A. If you have sufficient estimated object LOC and actual new and changed LOC data (three or more points that correlate), use procedure 4A
B. If you do not have sufficient estimated object LOC data but have sufficient estimated new and changed LOC data (three or more points that correlate), use procedure 4B
C1. If you have some estimated object LOC data and they are insufficient or do not correlate, use procedure 4C1.
C2. If you have some estimated new and changed LOC data and they are insufficient or do not correlate, use procedure 4C2.
D. If you have no historical data, use procudure 4D
4A The Size Estimating Procedure 4A
1. Using the probe tool, calculate the B0 and B1 regression parameters from the estimated object LOC and actual new and changed LOC.
2. If B0 is not near 0 (substantially smaller than the expected size of the new program), or B1 is not reasonably near 1.0 (between about 0.5 and 2.0), use procedure 4B.
4B The Size Estimating Procedure 4B
1. Using the probe tool, calculate the B0 and B1 regression parameters from the estimated new and changed LOC and actual new and changed LOC.
2. If B0 is not near 0 (substantially smaller than the expected size of the new program), or B1 is not reasonably near 1.0 (between about 0.5 and 2.0), use procedure 4C.
4C1 The Size Estimating Procedure 4C1 If you have any data on estimated object LOC and actual new and changed LOC, use the probe tool to set B0=0 and B1=(total new and changed LOC to date / total estimated object LOC to date).
4C2 The Size Estimating Procedure 4C2 If you have any data on estimated new and changed LOC and actual new and changed LOC, use the probe tool to set B0=0 and B1=(total new and changed LOC to date / total estimated new and changed LOC to date).
4D The Size Estimating Procedure 4D If you have no historical data, use your judgement to estimate the new and changed LOC from the estimated object LOC.
5 Size Prediction Intervals
  • If you used the regression method, calculate the prediction interval for the estimate using the probe tool.
  • if you did not use the regression method, estimate the maximum and minimum development times from your maximum and minimum productivity for prior programs.
