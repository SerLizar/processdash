Install Process Dashboard

Run the installation program by following the steps described below:

Installing on Windows

  1. If you have a CD-ROM containing the software, insert it into your computer. The installation process should start automatically. If it does not start automatically, double-click the file setup.exe on the CD-ROM. If you have a ZIP file containing the software, unzip it and double-click the extracted file setup.exe.
     
  2. The Process Dashboard requires a recent version of the Java Runtime Environment (JRE) to operate. If the JRE is not already installed on your computer, it will be installed. Choosing the default values at all prompts is recommended.
     
  3. The Process Dashboard installer will open:

  4. You may accept the default values for each prompt. When you reach the page asking which packs you wish to install, team leaders and process mentors should select the "Tools for Team Leaders" option. Individual team members do not need to select this option.

  5. The installer will allow you to select a directory where personal metrics data will be stored. You may accept the default value. Note: if you are upgrading the dashboard from an earlier version, and you previously chose to place your personal data in some other location, it is important to enter the true location of your personal data in this field. If you neglect to do this, you can run the installer a second time to set the data location correctly.
     
  6. Team leaders and process mentors will be prompted to identify a directory where team configuration information will be stored. Choosing a directory on a shared network server is generally required. The team leader or program manager should identify a shared network directory that all individuals can access. Then, the program manager and all team leads should select that same directory at this prompt.
     
  7. At the end of the installation, you will be given the opportunity to create shortcuts for starting the Process Dashboard. Click the Create Shortcuts button.

    This will create application shortcuts for your use:
    • A Process Dashboard shortcut will always be created, which allows you to manage your own personal data. (Even team leaders and process mentors will typically have some project tasks assigned to them; they should use this personal instance of the dashboard to perform that work.)
    • Team leaders and process mentors will also receive a Team Dashboard shortcut, which allows you to manage rolled-up team data. Clicking this shortcut will launch a team instance of the dashboard.
    These two shortcuts launch dashboard instances which are independent of each other. It is important to be cognizant of the difference between these two instances.

Installing on Mac OS X

Installing the Process Dashboard on Mac OS X requires a slightly different process.

  1. If you received a ZIP file of the Process Dashboard software, unzip it. Find the file called pdash-install-team-#_#_#.jar (it will be in a data subdirectory; the file will contain an actual version number rather than #_#_#). Alternately, you may have downloaded this JAR file directly; if that is the case, you can skip this unzip step.
     
  2. Double-click the pdash-install-team-#_#_#.jar file to begin the installation process.
     
  3. Follow the installer directions as described in the section above.
     
  4. An icon will be created in the Applications folder for the Process Dashboard. (If you chose to install the Tools for Team Leaders option, a Team Dashboard shortcut will be created as well.)

Installing on Unix/Linux

Installing the Process Dashboard on Unix requires a few additional steps.

  1. Download the Java Runtime Environment (JRE) from Sun Microsystems by visiting their Website http://java.sun.com/;. Install the software. Edit your executable path to include the java program.
     
  2. Install Firefox on your system if it is not already installed. Edit your executable path to include the firefox program.
     
  3. Either insert the CD-ROM into your computer, or unzip the ZIP file containing the software. Find the file called pdash-install-team-#_#_#.jar (it will be in a data subdirectory; the file will contain an actual version number rather than #_#_#). At a shell prompt, change to the directory containing that file, and type: 
    java -jar pdash-install-team-*.jar
  4. The Process Dashboard installer will appear.
     
  5. Follow the installer directions as described in the section above.
     
  6. Due to the variety of window managers on Unix platforms, the Process Dashboard installer cannot automatically create icons/shortcuts that you can use to run the program. Although icons/shortcuts are not created, the installer will create draft shell scripts that you may use to launch the Process Dashboard. In the directory you designated as your data directory (defaults to ~/.pspdata), you will find the file run-dash.sh. In the directory you designated as your team configuration directory, you will also find a file called run-dash.sh. (These launch two different dashboard instances; see the section above for an explanation of the two instances.) Review the contents of these scripts; in particular, you may wish to uncomment and edit the line that adds java and firefox to the executable path. For convenience, you may wish to manually create icons/shortcuts for launching the dashboard.
