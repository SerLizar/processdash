Inspection Instructions

There are several levels of inspection that need to be undertaken for the process scripts.

          Spelling errors and agreement with text of Humphrey's scripts
          Appearance in both IE and NS on WinNT, Win9X, and Linux (as appropriate)
          Consistency of linkage between master scripts and the subsequent scripts (e.g. HTML links from script.htm to develop.htm)
          All internal HTML targets need to exist and be correct
          Every formula in the global dataFile.txt needs to be checked against the correct calculations as given in the Humphrey text.
          Every formula in the sub-dataFile.txt's needs to be checked against the correct calculations as given in the Humphrey text.
          Every data entity in the HTML and JS files needs to be checked for correct spelling/case agreement with the dataFile.txt's
          All references to the current PSP level number should be correct (lots of copy-n-change took place)
          All uses of the <span> tag in order to show highlighting for the various types of text need to be verified
          Redundant equations in dataFile.txt not present in HTML scripts
          Check RE syntax to ensure it matches exactly the expected data elements
          Does equation use LOC when it should use New & Changed LOC
          &nbsp; on every empty table cell in tables that use <TABLE BORDER>
          hiddenData appropriately lists names of data elements that exist for internal use only
          Queries are sent to CGI scripts in the appropriate places
          Dollar signs ($) in RE's (not necessary at end anymore)
          {Completed} matching tag used appropriately
