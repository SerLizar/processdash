PSP(SM) Scripts/Forms Not Installed

PSP process definitions for use with the dashboard are available at no cost, but regrettably, these process definitions must be distributed as a separate download from the Process Dashboard. (This change is effective beginning in version 1.6; for more information, click here.) During installation, you chose not to install these process definitions. As a result, the PSP process scripts and forms are not available.

If you decide in the future that you would like to use these process definitions, run the Process Dashboard installer program again, and opt to install the PSP package(s).
