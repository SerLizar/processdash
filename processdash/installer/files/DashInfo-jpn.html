[ Process Dashboard ]  
 
  Process Dashboardとは？  
  　Process Dashboardは、GNU一般公衆利用許諾契約書に基づいて提供される、 強力かつ柔軟なソフトウェア・プロセス自動化ツールです。 Process Dashboardはメトリクスの収集と解析を自動化、簡素化し、 Personal Software Process(SM)のような高成熟度化したプロセスを、 より利用しやすくします。

　定義されたプロセスをとくに利用していない日々のタスクについても、 Process Dashboardによって高成熟度化した作業規範をそれらのタスクに適用する ことができます。 Dashboardは、例えば「タスク・スケジュール」機能のような、 日々の仕事の計画と、進捗の追跡をより効率的にするツールを提供します。

  
 
  どのような動作をするのでしょうか？  
  　Process Dashboardにはプロセスエンジンが組み込まれています。 プロセスエンジンはメトリクスデータと、プロセス・スクリプト、 帳票、およびその他のドキュメントとの統合化を、過去最高の水準で行います。 プロセスエンジンは簡単なXML、HTML、テキスト形式で記述されたプロセス定義を読み込み、 個人がプロセスを利用するためのユーザーインタフェイスを提供します。

　プロセス定義は固定的にプログラミングされているのではありませんので、 誰もが新しいプロセス定義を作成し、Dashboardとともに利用することができます。 既存のプロセス定義を利用したいのであれば、そのプロセス定義もまたインストールして利用することが可能です。

　Process DashboardはJava(TM)プログラム言語で記述されていますので、 Windows ®、Unix、Linux(TM)、Mac OS ® X などの、多くのプラットフォーム上で動作します。
