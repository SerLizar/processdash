Thank you for using the Process Dashboard!

If this is your first time to run the dashboard, you might benefit from the quick start guide, which provides a quick walkthrough of dashboard usage.

The first step to using the dashboard is to create a hierarchical work breakdown structure describing the work you do. To create this hierarchy, follow these steps:

  1. On the main dashboard window, click the "C" Menu, and choose "Hierarchy". The Hierarchy Editor window will appear.
     
  2. Build your hierarchy using commands in the "Edit" menu. Your hierarchy can be built from
    • "Nodes" (which are like folders on your hard drive) and
    • Templates (which are defined processes for completing a task - for example, PSP2.1).
    In general, you will want to create a hierarchy of Nodes describing the various projects/subprojects you support, then add templates underneath these nodes describing the lowest-level tasks to be done.
     
  3. You don't have to create your complete work breakdown structure right now - you can always come back and add nodes and templates to your hierarchy later. When you have finished editing your hierarchy, choose "Save" from the "File" menu.
